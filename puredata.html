Instalar Pure Data

Para las versiones extendidas-Pd, utilice el repositorio apt.puredata.info. Para añadirlo, ejecute estos comandos en el terminal:

sudo add-apt-repository "deb http://apt.puredata.info/releases` lsb_release-c | awk '{print $ 2}' `main"

sudo apt-key adv - keyserver keyserver.ubuntu.com - 9f0fe587374bbe81-recv clave

sudo apt-key adv - keyserver keyserver.ubuntu.com - D63D3D09C39F5EEB-recv clave

sudo apt-get update

sudo apt-get install pd-extendida



Ahora también puede encontrar el paquete pd-extendido en sináptica o Ubuntu Software Center.



Obtención de Bibliotecas individuales


También es posible conseguir empaquetados individualmente bibliotecas que se incluyen en los repositorios oficiales, pero construidas para versiones anteriores.Estos paquetes están en el eighthave / LIBDIRS PPA en launchpad.net. Algunos de los paquetes se remontan tan lejos como Ubuntu / kármica 09.10.


Conseguir la Build Beta extendido-Pd


También puede utilizar un depósito para conseguir las compilaciones anteriores de Pd-extended utilizando la PPA extendido-Pd. Para añadirlo, ejecute estos comandos en el terminal:

sudo add-apt-repositorio ppa: eighthave / extendidos-pd

sudo apt-get update

sudo apt-get install pd-extendida




Adición apt.puredata.info manualmente-Pd


Usted puede agregar esta línea al final de / etc / apt / sources.list O bien, puede añadirla en el programa de administración GUI de Synaptic (enConfiguración -> Repositorios menú, entonces el Software de terceros pestaña). Seleccione la línea para su versión de Debian o Ubuntu entre las siguientes opciones:

# Debian / antigua estable
deb http://apt.puredata.info/releases principal antigua estable

# Debian / estable
deb http://apt.puredata.info/releases principal estable

# Debian / testing
deb http://apt.puredata.info/releases prueba principal

# Debian / Sarge
deb http://apt.puredata.info/releases principal sarge

# Debian / etch
http://apt.puredata.info/releases deb graban principal

# Debian / Lenny
deb http://apt.puredata.info/releases principal lenny

# Debian / squeeze
http://apt.puredata.info/releases deb apretón principal

# Debian / sibilante
deb http://apt.puredata.info/releases principal sibilante
