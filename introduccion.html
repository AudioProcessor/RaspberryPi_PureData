Introducción

En este manual encontrarás todo lo referente a la configuración y el trato de sonido mediante un Raspberry Pi.
Se utiliza distintos componentes tanto de hardware como de software. Todo fue investigado de manera cuidadosa, por lo que esta manual será una guía sencilla para mostrarte como realizar un procesador de efectos musicales utilizando una Raspberry Pi, Pure Data y OSC.
